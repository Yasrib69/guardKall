# Sponsor integration mapping

## DigitalOcean (Best Use of DigitalOcean)
- Deploy **telephony switchboard** on a Droplet.
- Handles Twilio webhooks + Media Streams WebSocket.

## Google Cloud / Gemini (Best Use of Gemini API)
- **Gemini 1.5 Pro** handles real-time scam detection.
- Use Google AI Studio for prompt iteration and latency tuning.

## ElevenLabs (Best Use of ElevenLabs)
- Generate the **Counter-Agent** voice on takeover.
- Stream or return short clips for Twilio playback.

## Snowflake (Best Use of Snowflake API)
- Store **Scam Signatures** (hashes / embeddings).
- Query before takeovers to identify known scripts.

## OpenRouter (resource credits)
- Optional routing for model experiments.
- Use as a fallback when testing alternative models.

## DigiKey (optional hardware)
- Raspberry Pi demo: physical “home phone” version of Guardkall.
