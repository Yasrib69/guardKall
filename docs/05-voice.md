# Voice service (ElevenLabs)

## Persona
"Confused Bureaucrat" - polite, slow, repetitive, asks for verification details and forms.

## Inputs
- Short response text from telephony service (or scripted phrases)

## Output
- Streaming audio or downloadable clip compatible with Twilio playback

## TTS hints
- Use a consistent voice ID
- Insert hesitations and clarifying questions to maximize scammer time

## Example prompts
- "I'm going to need the reference number on your Form 27B... could you repeat it?"
- "Sorry, the line cut out. Can you repeat that from the start?"
